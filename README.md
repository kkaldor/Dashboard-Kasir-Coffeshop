# Kasir Web Dashboard (Prototype)

Prototype dashboard kasir untuk coffeshop berbasis web.

## Fitur
- Pilih produk + qty
- Keranjang (tambah / tambah qty / kurangi / hapus)
- Ringkasan total (subtotal/diskon/pajak dummy)
- Pembayaran (cash/qris/transfer)
- Riwayat transaksi hari ini (tersimpan `localStorage`)
- KPI sederhana (total hari ini, jumlah transaksi, item terjual)

## Cara Jalankan
Buka file:
- `index.html`

Paling mudah:
- Double click `index.html`, atau
- Serve lewat extension VSCode Live Server.

## Catatan Lanjutan
Untuk versi produksi biasanya ditambah:
- Backend (Node/Express atau Laravel)
- Database (MySQL/PostgreSQL)
- Manajemen produk & stok
- Role kasir/manajer
- Export laporan (PDF/Excel)
